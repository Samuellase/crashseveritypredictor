# Vehicle Crash Severity Predictor

Binary classification system that predicts whether a US traffic accident will result in **low** (severity 1–2) or **high** (severity 3–4) impact on traffic flow, using weather, road infrastructure, and temporal features.

## Results

| Model | Test Accuracy | AUC |
|---|---|---|
| Baseline Random Forest | ~78% | — |
| Tuned Random Forest | ~79% | — |
| XGBoost | ~79% | — |
| **Stacking Ensemble** | **~80%** | **0.749** |

The stacking ensemble (Random Forest + XGBoost + Decision Tree → Logistic Regression meta-learner) achieved the best generalization performance.

## Dataset

[US Accidents (2016–2023)](https://www.kaggle.com/datasets/sobhanmoosavi/us-accidents) — ~500,000 records sampled from 2.8M US accident records. Class-balanced via undersampling before training.

> The raw dataset is not included in this repo due to size. Download it from Kaggle and place it at `data/raw/US_Accidents_March23.csv`.

## Features

**Raw features:** temperature, humidity, pressure, visibility, wind speed, precipitation, weather condition, road infrastructure (junction, traffic signal, stop, crossing), time of day, day of week, month.

**Engineered features:**
- `Weather_Severity` — composite score from precipitation, visibility, wind speed
- `Rush_Hour` — flag for 7–9am and 4–6pm windows
- `Dangerous_Hours` — flag for 10pm–4am
- `Infrastructure_Risk` — count of road hazard features at crash location
- `Temp_Humidity_Index` — interaction between temperature and humidity
- `Weekend_Night` — combined weekend and late-night flag

## Methodology

- **Class balancing:** Undersampling majority class to address ~63/37 class imbalance
- **Hyperparameter tuning:** RandomizedSearchCV with 3-fold CV on Random Forest
- **Ensemble stacking:** Base learners (RF, XGBoost, Decision Tree) feed into Logistic Regression meta-learner
- **Evaluation:** Accuracy, Precision, Recall, F1, ROC-AUC, Confusion Matrix
